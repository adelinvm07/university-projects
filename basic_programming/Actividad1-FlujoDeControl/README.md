# Actividad 1 - Cálculo del ISR (Impuesto Sobre la Renta)

## Descripción 
Programa que pide al usuario su sueldo mensual para calcular el ISR correspondiente a dicho salario. Esto acorde al código tributario (Ley 11-92, Art. 327) de la República Dominicana en el año 2026.
